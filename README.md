# Machanor — From Vision to Production

Light-theme marketing site for Machanor, an AI-native digital transformation consultancy.

## What's here
- `index.html` — page structure
- `styles.css` — design system (light theme, ink/paper palette, blue + gold accents)
- `script.js` — canvas particle network (hero + CTA), magnetic buttons, scroll reveals, animated counters

## Deploy to Vercel
1. Push this folder to a GitHub repo
2. Import the repo in Vercel (vercel.com/new)
3. Framework preset: "Other" — no build step required
4. Deploy

Or via CLI:
```
npm i -g vercel
vercel
```

## Local preview
```
npx serve .
```
